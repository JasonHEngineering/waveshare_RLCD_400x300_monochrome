# Waveshare 4.2" RLCD - Monochrome Smart Clock


<img width="1412" height="994" alt="image" src="https://github.com/user-attachments/assets/17733e5d-e06e-40ba-b1f4-c91c242011e1" />


Smart clock using a ESP32-S3-RLCD-4.2 from Waveshare which is is a RLCD AIoT development board based on ESP32-S3, supporting Wi-Fi and BLE dual-mode communication. "Smart" features includes auto-sync of internet date/time, get buses arrival time, display random trivia, and ability to query from ChatGPT via audio commands. 

Note that some of the contents are catered to that of Singapore's, such as date/time, temperature, humdity, weather and index PM2.5.

This is a programming package for a Waveshare ESP32-S3-RLCD-4.2. This is a monochrome Reflective LCD with no grey scale, which utilize surrounding light for the display. The contrast is in fact better in strong sunlight. In terms of power consumption, this is consuming more power than that of e-Ink, however, less power than that of traditional LCD. Overall, this is a very good display for balancing power consumption in outdoor use conditions.

The cons is mainly on the need to connect to WiFi for most of the functions here.
