This V1 is for embedded passords and apiKeys in FW itself. Therefore wifi SSID, passwords, bus stop IDs and apiKeys needs to be written into code itself.
This does means that whenever an item above was changed, one has to edit and upload the firmware again.

On the other hand, if one desire to have a fixed code loaded and change the above configuration only by SD card update, refer to V2 instead.
